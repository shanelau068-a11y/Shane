# 阿彦的消防车绘画小程序

这是一个给幼儿园大班小朋友使用的网页小程序，主题是“学习画消防车”。

## 功能

- 分步骤教学：6 步画出消防车。
- 儿童友好画板：支持颜色、画笔粗细、撤销、清空。
- 轮廓辅助：可一键显示/隐藏消防车参考轮廓。
- 个性化：页面文案中包含孩子名字“阿彦”。
- 作品墙：支持从手机相册上传多张作品，自动保存到本机浏览器（localStorage）。

## 本地运行

直接双击 `index.html` 即可打开。

或使用一个本地服务器：

```bash
python3 -m http.server 8000
```

然后访问：<http://localhost:8000>

## 手机上怎么打开？

### 方式 A：同一个 Wi-Fi 下临时打开（适合家里自己用）

1. 在电脑进入项目目录并启动：

   ```bash
   python3 -m http.server 8000
   ```

2. 查看电脑的局域网 IP（例如 `192.168.1.23`）。
3. 手机和电脑连同一个 Wi-Fi。
4. 手机浏览器输入：`http://192.168.1.23:8000` 即可打开。

> 这个方式只能在同一网络内使用，发给外网朋友通常打不开。

### 方式 B：部署到公网（适合发给微信好友）

要让朋友在微信里直接点开，需要一个公开的 `https://` 链接。推荐以下任意一种：

- GitHub Pages（免费，稳定）
- Vercel（免费，部署快）
- Netlify（免费，操作简单）

部署完成后会得到类似 `https://xxx.vercel.app` 的地址，把这个地址发到微信，对方点击即可使用。

## 微信里能不能直接打开？

- ✅ 可以：发 **公网 HTTPS 链接**，微信内可直接打开。
- ❌ 不可以：直接发 `index.html` 文件让对方点开（微信不会当网页运行本地文件）。

## 给朋友使用的最简步骤（推荐）

1. 把本项目上传到 GitHub。
2. 用 Vercel 一键导入并部署（默认就会给 HTTPS 链接）。
3. 把链接发给朋友微信。
4. 朋友点击后，手机里直接使用“阿彦的消防车绘画小课堂”。

## 怎么把孩子画的图加进去？

1. 打开页面后，找到“阿彦作品墙”。
2. 点击“📷 选择图片”，从手机相册选中一张或多张作品图。
3. 上传后会自动展示在作品墙，且会保存在当前手机浏览器里。
4. 点击“清空作品墙”可以一次清除。
