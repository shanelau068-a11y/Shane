# 图形化小游戏：接金币（键盘 + 鼠标）

这是一个使用 Python `tkinter` 写的图形化小游戏，不需要额外安装第三方库。

## 玩法

- 你控制底部蓝色方块去接掉落的金币。
- 每接到一个金币，分数 +1，金币下落会稍微变快。
- 游戏时长 30 秒，结束后显示最终分数。

## 操作方式

- **键盘**：
  - `←` / `→` 移动
  - `A` / `D` 也可移动
  - `Enter` 或 `Space` 开始
- **鼠标**：
  - 点击“鼠标点击重开”按钮可重开一局

## 运行

```bash
python3 game.py
```

> 如果在无图形界面的终端环境运行（如纯服务器），窗口可能无法弹出。
