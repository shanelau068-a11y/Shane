# 衣柜拉手高清替换 APP

一个面向衣柜门拉手电商出图的纯前端工具。商家可以上传柜子样板图和拉手产品图，拖拽定位框覆盖原拉手位置，实时预览替换效果，并导出无水印 3:4 高清 PNG 主图。

## 功能

- 上传柜子样板图片与拉手产品图片。
- 在画布上拖动定位框或四个角点，精准指定原拉手区域。
- 自动去除拉手白底，透明 PNG 会保留原透明度。
- 可选智能覆盖原拉手阴影，减少原图拉手残留。
- 支持旋转、缩放新拉手，适配不同柜门角度和尺寸。
- 导出 1500×2000、2250×3000 或 3000×4000 的 3:4 无水印 PNG 图片。

## 本地运行

```bash
npm start
```

然后打开 <http://127.0.0.1:4173/>。

## 检查

```bash
npm run check
```
