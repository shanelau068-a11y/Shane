    第三部分：绘画小口诀

    • 先方后圆：先画车身和梯子，再画车轮。
    • 先大后小：先画大轮廓，再画门把手、车灯。
    • 慢慢画线：画得慢一点，线条会更稳。
    • 多夸自己：每画完一次都说“阿彦真棒！”

    手机打开与微信分享

    1. 先把这个页面部署到公开网址（HTTPS），比如 GitHub Pages、Vercel 或 Netlify。
    2. 在手机微信里把网址发给朋友，对方点开就能直接使用。
    3. 不能直接把 index.html 文件发到微信里打开，必须先变成在线链接。
